# Food Delivery App

## Краткое описание идеи
Платформа для заказа еды из ресторанов с мобильным приложением, веб-сайтом и серверной частью.

## Структура репозитория
- **MobileApp** — клиентское мобильное приложение
- **ServerApi** — серверное API (бэкенд)
- **Web** — веб-сайт для заказов
