# Серверное API

## Что будет в этой папке
Код серверной части (бэкенд), реализующий REST API.

## Планируемые компоненты/модули
- Модуль аутентификации (JWT)
- API для работы с ресторанами и меню
- API для оформления и отслеживания заказов
- Административная панель (управление заказами)
- Интеграция с платёжным шлюзом

## Взаимодействие с другими частями
- ServerApi обрабатывает запросы от MobileApp и Web.
- Взаимодействует с базой данных.
