# Мобильное приложение

## Что будет в этой папке
Исходный код клиентского мобильного приложения для iOS/Android (React Native / Flutter).

## Планируемые компоненты/модули
- Экран авторизации и регистрации
- Главный экран с меню ресторанов и блюд
- Экран оформления заказа
- Экран истории заказов
- Модуль уведомлений

## Взаимодействие с другими частями
- MobileApp отправляет запросы к ServerApi для получения данных (меню, оформление заказа).
- Web-версия использует те же API, но с адаптацией под большие экраны.
