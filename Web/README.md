# Веб-сайт

## Что будет в этой папке
Исходный код веб-приложения (React / Vue).

## Планируемые компоненты/модули
- Страница входа и регистрации
- Каталог ресторанов и блюд
- Корзина и оформление заказа
- Личный кабинет с историей заказов
- Адаптивный дизайн

## Взаимодействие с другими частями
- Web взаимодействует с ServerApi через HTTP-запросы.
